## üì• Setup Instructions

### 1. Database Configuration
Run the contents of `SETUP.sql` in your **Supabase SQL Editor**. This will create the following tables:
- `site_context`: Stores brand voice and target audience.
- `agent_logs`: Full audit trail of AI actions.
- `agent_memories`: Persistent state for complex LangGraph flows.
- `learning_data`: Performance metrics for AI learning.

### 2. Environment Variables
Add the following to your `.env` file (see `.env.agentic` for template):
```bash
# Agents Layer
OPENAI_API_KEY=sk-...
GOOGLE_GENAI_API_KEY=...
AGENT_MODEL=gpt-4-turbo # Options: gpt-4-turbo, gemini-pro
AGENT_AUTONOMOUS_MODE=false # Set to true for weekly auto-optimization
```

### 3. Using the AI Assistant
1. Go to the **Admin CMS Dashboard** (`/secure-access/admin/cms`).
2. You will see the **Agentic AI CMS Core** command bar at the top.
3. Type a goal, e.g., *"Improve the homepage for conversion"* or *"Setup a new services page for Yoga classes"*.
4. Watch the agent plan and execute the changes using the site's tool ecosystem.
